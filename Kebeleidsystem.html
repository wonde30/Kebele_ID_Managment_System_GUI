01 Description

The Kebele ID Management System is a Java-based desktop application designed to digitize and streamline the management of resident identification records in kebele offices. The system replaces manual paper-based record keeping with a secure, role-based digital solution supporting ID card generation, approval workflows, and data reporting. All data is stored locally using SQLite — no internet connection required.

03 Features
F-01 Resident Record Management
  • Add, edit, delete, and search resident records
  • Full profile with personal and address information
F-02 ID Card Generation
  • Automatic ID card creation with embedded QR code
  • QR code links to resident record for instant verification
F-03 Role-Based Access Control — 5 User Roles
ROLE ACCESS LEVEL PERMISSIONS
ADMIN Full access System config, user management, all modules
SUPERVISOR Management Resident management, ID generation, reports
STAFF Operational Add/edit residents, generate ID cards
DATA ENCODER Data entry Add and edit residents only
VIEWER Read-only View own linked record only
F-04 Approval Workflow
  • New registrations require approver sign-off before activation
  • Prevents unauthorized or duplicate entries
F-05 Reports & Data Export
  • Export records to PDF using iText library
  • Export records to Excel using Apache POI
  • Summary reports for registered residents
F-06 Activity Logging
  • All user actions recorded with timestamp and user ID
  • Full audit trail for accountability and review
